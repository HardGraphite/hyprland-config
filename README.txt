Hyprland configuration
======================

Dependencies
------------

- waybar           status bar
- fuzzel           program menu & dmenu
- mako             notification daemon
- wl-clipboard     cli clipboard tool
- clipman          clipboard manager
- swaybg           wallpaper
- swaylock         screen lock
- swayidle         wait and lock
- qt5ct            Qt configuration
- fcitx5           input method

Optional dependencies
---------------------

- grim             screenshot tool
- slurp            select screen region, can work with grim
